## 🚀 Como Começar (3 Passos)

### Passo 1: Preparar Supabase (5 min)
```bash
1. Acesse https://supabase.com
2. Crie projeto "neocanteiro"
3. Copie credenciais (URL, Keys)
4. Crie .env.local com as variáveis
```

### Passo 2: Criar Schema (2 min)
```bash
1. Supabase: SQL Editor
2. Cole DATABASE_SCHEMA.sql
3. Execute (Run)
4. Crie 2 buckets: diarios, avatares
```

### Passo 3: Rodar Local (1 min)
```bash
npm install @supabase/supabase-js
npm run dev
→ http://localhost:3000
```

**Total: 15 minutos até ter sistema funcionando!**
