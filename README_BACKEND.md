## 🚀 3 Passos Para Começar

### 1️⃣ Setup Supabase (5 min)
```
→ supabase.com → Novo projeto
→ Copiar credenciais
→ Criar .env.local
```

### 2️⃣ Criar Schema (2 min)
```
→ SQL Editor: Cole DATABASE_SCHEMA.sql
→ Storage: Crie 2 buckets (diarios, avatares)
```

### 3️⃣ Rodar Local (1 min)
```
npm install @supabase/supabase-js
npm run dev
→ http://localhost:3000
```
